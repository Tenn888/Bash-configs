# Bash-configs

Этот репозиторий содержит конфигурации **Bash** и **Starship** для обычного пользователя и Root.  
Сделано для личного использования.

---

## 📂 Структура репозитория

Bash-configs/
├── user/
│ ├── bashrc
│ └── config/starship.toml
├── root/
│ ├── bashrc
│ └── config/starship.toml
├── install.sh
└── README.md


---

## ⚡ Установка на новой системе

1. Клонируем репозиторий:

```bash
git clone https://github.com/Tenn888/Bash-configs.git ~/Bash-configs
cd ~/Bash-configs
```

2. Делаем скрипт исполняемым и запускаем:

```bash
chmod +x install.sh
./install.sh
```
